Github Notification Feed

Github provides an RSS feed for your home page - scroll to the bottom and you'll find the link - but doesn't provide one for your actual notification. This app fills that gap by serving your notification feed as an ATOM feed.

To get your own feed, you need to first make a Personal Access Token with the notifications scope. Then, your feed will be at

https://gh-notifeed.herokuapp.com/username/token/.atom

(replace username with your github username and token with the Personal Access Token you just generated)

Made by ralismark. Source code for this on github.
